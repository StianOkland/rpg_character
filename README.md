# RPG Character
RPG Character classes (Mage, Ranger, Rogue and Worrier) in C#.

## Instalation
Clone the project and import classes to use.

```bash
git clone
```

## Tests
Clone the project and run tests in Microsoft Visual Studio created in RPGCharacterTests folder.
